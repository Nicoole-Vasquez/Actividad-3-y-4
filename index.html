Tabla en HTML Las tablas en HTML son utilizadas para organizar y presentar datos en un formato tabular. La estructura básica de una tabla HTML consiste en una serie de filas y columnas que contienen datos o encabezados. Formulario en HTML Los formularios en HTML permiten a los usuarios enviar datos al servidor para procesamiento, usando la etiqueta form para definirlos.
Etiqueta TABLE La etiqueta table define una tabla en HTML. Todo el contenido de la tabla, incluidos los encabezados y los datos, se coloca dentro de esta etiqueta. Etiqueta FROM Esta etiqueta encapsula todos los elementos del formulario y define atributos como action (URL de destino) y method (GET o POST).
Atributos de TABLE Los atributos de table permiten configurar la apariencia y el comportamiento de la tabla. Algunos de los atributos comunes incluyen: Borde, Cellspacing, Celpadding, Width y Height. Etiqueta INPUT Se utiliza para crear diferentes tipos de campos de entrada, como texto, contraseñas, casillas de verificación y botones.
Etiqueta TR La etiqueta tr define una fila en una tabla. Cada fila puede contener una serie de celdas de datos (td o celdas de encabezado th. Etiqueta LABEL Asocia un texto descriptivo con un elemento de formulario, mejorando la accesibilidad al permitir que los usuarios identifiquen fácilmente los campos.
Atributo de TR Aunque tr no tiene atributos únicos que se utilicen comúnmente, puede utilizarse con atributos generales como class y id para aplicar estilos o identificaciones específicas. Etiquta SELECT Crea un menú desplegable con múltiples opciones, permitiendo a los usuarios seleccionar una opción de una lista.
Etiqueta TD y TH Td define una celda de datos en una tabla. Se utiliza para contener los datos en una fila. Th define una celda de encabezado en una tabla. El texto en las celdas th generalmente se muestra en negrita y centrado. Etiqueta TEXTAREA Define un campo de entrada de texto multilínea, útil para introducir grandes bloques de texto.
Atributos de TD y TH Los atributos de td y th permiten definir la apariencia y el comportamiento de las celdas Etiquetas FIELDSET, LEGEND Agrupan elementos de formulario relacionados bajo un conjunto definido, con legend proporcionando un título para el conjunto.
Tablas avanzadas Las tablas avanzadas en HTML permiten una mayor organización y presentación de datos utilizando etiquetas como thead, tfoot y tbody, que ayudan a estructurar el contenido de la tabla. Etiqueta Meter Representa una medida escalar en un rango conocido, útil para mostrar niveles o valores proporcionales como un medidor de capacidad.
Etiqueta THEAD La etiqueta thead agrupa los encabezados de una tabla. Este grupo de encabezados puede ser estilizado de forma independiente y se encuentra al inicio de la tabla. Etiquta PROGRESS Muestra el progreso de una tarea, como una barra de progreso, indicando cuán cerca está la tarea de completarse.
Etiqueta TFOOT La etiqueta tfoot agrupa los pies de página de una tabla. Este grupo se coloca al final de la tabla y puede contener información resumida o de pie de página. Etiquta DETALIST Proporciona una lista de opciones predefinidas para un input, permitiendo selección rápida o introducción manual.
Etiqueta TBODY La etiqueta tbody agrupa el cuerpo principal de los datos de la tabla. Los datos de la tabla se colocan dentro de esta etiqueta, separando claramente el contenido de los encabezados y los pies de página. Etiqueta SUMMARY Proporciona un encabezado interactivo para el elemento details, que puede ser expandido para revelar información adicional.

Tema 2. Define los elementos que componen la estructuración de una página web:

  1. Contenedor (wrapper):

    2. Un contenedor o wrapper es un concepto que aparece en distintos contextos de programación y teoría de la computación, con diferentes propósitos pero una idea común: encapsular o envolver algo para dotarlo de propiedades adicionales, facilitar su manipulación, o abstraer detalles internos.

  2. Cabecera (header):

    3. Una cabecera (header) es una sección inicial en un archivo, mensaje, documento o estructura que contiene metadatos o información clave para interpretar o procesar el contenido principal.

  3. Contenido (content):

    4. El contenido (content) es la información o los datos principales que constituyen el núcleo de un mensaje, archivo, documento o estructura, diferenciándose de los elementos auxiliares como cabeceras o metadatos.

  4. Menú (menu):

    5. Un menú (menu) es una lista estructurada de opciones o comandos que permite a los usuarios interactuar con un sistema, aplicación o dispositivo para ejecutar acciones o acceder a diferentes secciones.

  5. Pie (footer):

    6. Un pie (footer) es la sección inferior de un documento, página web o interfaz que suele contener información complementaria, como créditos, enlaces, copyright o datos de contacto.

  6. Lateral (sidebar):

    7. Un lateral (sidebar) es una sección secundaria ubicada a un lado de una interfaz o página web, que contiene información adicional, menús, herramientas o enlaces complementarios al contenido principal.

  7. Etiqueta div:

    8. Una etiqueta div en HTML es un contenedor genérico que se utiliza para agrupar elementos y aplicarles estilos o funcionalidades mediante CSS y JavaScript, sin representar ningún significado semántico por sí misma.
